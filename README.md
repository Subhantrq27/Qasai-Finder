# Qasai Finder

Pakistan's first online butcher directory — Find available & booked qasai in your city!

**Live Site:** [qasaifinder.netlify.app](https://qasaifinder.netlify.app)

---

## Features

- Search qasai by name, shop, or area
- City filter — Lahore, Islamabad, Attock
- Status — Available, Partial, Booked
- Call & WhatsApp button on every card
- Google Form registration for qasai
- Admin Panel — Add, update & delete qasai
- Real-time Firebase Firestore database
- Live stats — Total, Available, Booked count
